# Flexacil Perú

> Suplemento articular formulado para el mercado peruano con glucosamina, cúrcuma y plantas nativas andinas.

**[Visitar flexacil.pe](https://flexacil.pe)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Flexacil Perú](https://flexacil.pe), un recurso de salud basado en evidencia científica enfocado en **Articulaciones**.

### Ingredientes clave

Glucosamina sulfato, curcumina, condroitina, uña de gato, colágeno hidrolizado

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
